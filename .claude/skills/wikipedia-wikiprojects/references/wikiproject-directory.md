## Common Banner Parameters

WikiProject banners are placed on article talk pages. Key parameters:

| Parameter | Purpose | Common Values |
|-----------|---------|---------------|
| `class` | Quality rating | `FA`, `FL`, `GA`, `B`, `C`, `Start`, `Stub`, `List`, `NA` |
| `importance` | Priority within project | `Top`, `High`, `Mid`, `Low`, `NA` |
| `listas` | Sort key for category indexes | Surname, First (e.g., `Einstein, Albert`) |
| `attention` | Needs immediate review | `yes` |
| `needs-infobox` | Missing infobox | `yes` |
| `needs-photo` / `needs-image` | Missing image | `yes` |

**Quality scale reference:**

| Class | Criteria |
|-------|----------|
| **FA** | Featured Article — professional, comprehensive, well-written |
| **GA** | Good Article — meets GA criteria, useful to casual readers |
| **B** | Mostly complete, no major gaps, reasonable coverage |
| **C** | Substantial but missing important content or has style issues |
| **Start** | Developing but far from complete; reliable sources needed |
| **Stub** | Very short; a basic description; needs expansion |

## Subpage Naming Conventions

WikiProjects use a standard set of subpages:

```
Wikipedia:WikiProject_<Topic>/
├── Assessment          # Quality × importance matrix
├── Popular_pages       # Bot-generated pageview rankings
├── Article_alerts      # Automated AfD/PROD/GAN notices
├── Cleanup_listing     # Articles needing specific work
├── Members             # Editor participants
├── To_do               # Manual task list
├── Resources           # Reliable sources, style guides
└── Templates           # Project-specific templates
```

Not all projects maintain all subpages. Active projects typically have at
least `Assessment` and `Popular_pages`.

## Task Force & Work Group Patterns

Large WikiProjects commonly delegate to task forces:

```
Wikipedia:WikiProject <Parent>          ← Main project
├── /Task Force: <Sub-topic>            ← Task force
├── /<Sub-project>                      ← Semi-autonomous sub-project
└── /Work group: <Topic>                ← Work group (less formal)
```

**Banner syntax for task forces:**
```wikitext
{{WikiProject Military history|class=B|b1=y|b2=y|US-task-force=yes}}
{{WikiProject Biography|class=C|importance=Low|s&a-work-group=yes}}
```

The `*-task-force=yes` and `*-work-group=yes` parameters flag the article as
within a task force's scope. These are displayed in the banner shell.
