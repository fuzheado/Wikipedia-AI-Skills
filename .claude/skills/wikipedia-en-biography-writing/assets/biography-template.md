## Section Guidelines

| Section | Required? | Notes |
|---|---|---|
| Lead | Yes | 2-4 paragraphs. No citations needed if facts sourced in body. |
| Early life | Recommended | Skip if no reliable sources available. |
| Career | Yes | Core of the article. |
| Personal life | Optional | Keep minimal for BLPs. |
| Awards | If applicable | Bullet list or prose. |
| Selected works | If applicable | Numbered list or table. |
| References | Yes | Always include `{{reflist}}`. |

## Checklist Before Submitting

- [ ] Notability established in lead
- [ ] Every substantive claim has an inline citation
- [ ] No unsourced contentious content (especially for living subjects)
- [ ] NPOV — no promotional language
- [ ] NOR — no synthesis or original analysis
- [ ] Verifiability — all sources are reliable
- [ ] Infobox filled with only sourced fields
- [ ] Categories added (if known)
- [ ] Section order matches standard biography structure
