# Verification Protocol — wikipedia-en-article-audit

Extended guidance for the Phase 2 human reviewer.

## Source hierarchy

When two sources disagree, prefer in this order:

1. **Peer-reviewed academic journals** — highest reliability for factual claims
2. **Major news outlets with editorial oversight** — Reuters, AP, NYT, WSJ, Bloomberg
3. **Government publications and court records** — primary source data
4. **University presses and academic books** — Van Zandt, Tuttle Publishing
5. **Theses and dissertations** — use with caution; may contain original research
6. **Primary sources** — letters, logbooks, probate records, census data
7. **Wikipedia sister articles** — useful for cross-reference but not as primary citations
8. **Self-published sources** — blogs, personal websites — do not use for BLPs

## Handling conflicting sources

If Source A says "X" and Source B says "not X":

1. Check the dates — is one more recent?
2. Check the domain — does one source specialize in this topic?
3. Check the context — is the contradiction real or a misinterpretation?
4. If both are equally reliable and contradictory, report the dispute neutrally:
   "Source A claims X, while Source B claims Y."

## BLP-specific considerations

For living subjects (BLP=yes):

- **Contentious material** requires multiple high-quality sources
- **Remove immediately** any unsourced negative claim — do not leave it even temporarily
- **Private information** (address, phone, exact birth date without source) must be removed per WP:BLPPRIVACY
- **Epithets and labels** ("criminal," "fraud," "scandal") must be attributed to specific sources, not asserted by Wikipedia's voice
- **Harm principle** — if publishing a fact could cause real-world harm and it's not essential, omit it

## Working with the Wayback Machine

For inaccessible sources:

1. Check `https://web.archive.org/web/*/URL` for archived copies
2. If the URL has changed but the content moved, search for the title
3. For books, check Google Books previews and HathiTrust
4. For paywalled news, check if a wire service version exists (e.g., Reuters stories are often republished by multiple outlets)

## Assessing source reliability

| Source type | Reliability | Notes |
|---|---|---|
| Reuters, AP, AFP | High | Wire services with editorial oversight |
| NYT, WSJ, Bloomberg | High | Major newspapers with fact-checking |
| Regional newspapers | Medium-High | Good for local history (e.g., Newsday for Long Island) |
| Trade publications | Medium | Good for industry-specific claims (e.g., Electronics for You) |
| University repositories | Medium-High | Content is reliable but access may be restricted |
| Corporate bios | Low-Medium | Useful for basic facts (title, dates) but potentially self-serving |
| User-generated (Findagrave, Wikidata) | Low | Use as leads only, not as authoritative sources |
| Blogs, forums, social media | Do not use | Never acceptable for BLPs |
